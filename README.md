# ANN-SVD
ANN-SVD is a hybrid framework that integrates Artificial Neural Networks with Singular Value Decomposition (SVD) to accelerate PDE solving. SVD is applied to the weight matrix between the first and second hidden layers, using Thin, Compact, and Truncated SVD to improve convergence and efficiency.
